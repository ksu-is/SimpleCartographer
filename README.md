# SimpleCartographer
A program to create simple marks on a map through interaction with a GUI. 
## Project Goals

- Allow users to make marks on map though a GUI
- Download created maps as png or .svg file
- Ensure user flexibility in map creation

## Technologies Used

- py-staticmaps
- tkinter
